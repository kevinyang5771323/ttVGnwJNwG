# 社区疫苗接种管理系统 java176

## 项目概述

社区疫苗接种管理系统是基于SpringBoot框架开发的前后端分离系统。该系统旨在为社区提供便捷的疫苗接种预约管理功能，支持管理员与用户的不同角色操作。

## 技术栈

- 后端：SpringBoot、Mybatis-Plus、JPA
- 数据库：MySQL
- 前端：Vue、ElementUI

## 功能模块

### 系统角色

#### 管理员（Admin）

- 管理员管理
- 用户管理
- 管理新闻列表
- 论坛列表管理
- 疫苗信息管理
- 添加预约计划
- 疫苗入库与出库
- 添加接种记录

#### 用户（User）

- 查看疫苗信息
- 查看预约计划
- 预约接种疫苗
- 查看预约审核详情
- 查看接种记录

## 运行环境

- 操作系统：Win10/11
- 开发工具：IDEA IU 2022.3.3
- 开发语言：Java (JDK 8u371)、Vue.js (Node.js 13.14)
- 项目管理工具：Apache Maven 3.9.4
- 数据库：MySQL 8.0

## 部署步骤

1. 执行目录中的SQL文件
2. 解压项目压缩包
3. 使用IDEA导入项目
4. 等待Maven下载所需Jar包
5. 启动后端项目
6. 前端项目使用npm启动
